# snake_game
A cross-platform version of the classic snake game written in C/C++.


## Usage Instructions
### Linux
Clone into the repository using `git clone https://github.com/samrudhnelli/snake_game.git`.\
Get into the directory using `cd snake_game`.\
Setup execute permissions for the .sh file using `chmod +x snake.sh`.\
Run the snake_game using `./snake.sh`.

### Windows
Clone into the repository using `git clone https://github.com/samrudhnelli/snake_game.git` (or download the ZIP file and extract it).\
Get into the directory using `cd snake_game`.\
Run the snake_game by double-clicking the `snake.bat` file in your File Explorer, or execute it from the Command Prompt using `snake.bat`.
